# Evaluation Baselines

This directory stores baseline evaluation reports for regression detection.

## Usage

Baseline reports are JSON files generated by the evaluation system. They capture the expected behavior of scenarios at a specific point in time.

### Creating a Baseline

```bash
# Run evaluation and save as baseline
pnpm tsx scripts/benchmark-runner.ts --suite context --json
cp evals/reports/suite-context-*.json evals/baselines/context.baseline.json
```

### Comparing Against Baseline

```bash
# Run evaluation
pnpm tsx scripts/benchmark-runner.ts --suite context --json

# Compare with baseline
ccjk eval compare --baseline evals/baselines/context.baseline.json --candidate evals/reports/suite-context-*.json
```

## Baseline Naming Convention

- `{suite}.baseline.json` - Current baseline for a suite
- `{suite}.baseline.v{version}.json` - Version-specific baseline

## When to Update Baselines

Update baselines when:
- Intentional behavior changes are made
- Performance improvements are verified
- New scenarios are added

Do NOT update baselines to hide regressions.

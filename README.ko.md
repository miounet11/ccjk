```yaml
# 스킬은 작업에 따라 자동으로 활성화
triggers:
  - pattern: "*.test.ts"      # 테스트 중? 테스트 전략 로드
  - context: "git:staged"     # 커밋 중? 코드 리뷰 실행
  - project: "react"          # React 프로젝트? 베스트 프랙티스 로드
```

```bash
ccjk postmortem <action># 과거 버그에서 학습
ccjk skills <action>    # AI 스킬 관리
ccjk hooks <action>     # 라이프사이클 훅 설정
```

### 설정 명령어

```bash
ccjk config <action>    # 설정 관리
ccjk providers <action> # API 프로바이더 관리
ccjk permissions        # 권한 관리
ccjk sandbox <action>   # 샌드박스 환경 제어
```

### 시스템 명령어

```bash
ccjk system setup       # 초기 설정
ccjk system upgrade     # 모든 컴포넌트 업그레이드
ccjk system versions    # 버전 확인
ccjk check-updates      # 업데이트 확인
ccjk stats              # 사용 통계
ccjk session <action>   # 세션 관리
```

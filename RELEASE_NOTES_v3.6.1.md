## üöÄ Getting Started

### Installation

```bash
# Install or update to v3.6.1
npm install -g ccjk@3.6.1

# Verify installation
ccjk --version
```

### Quick Start

```bash
# Interactive setup (1 minute)
npx ccjk init

# Or use quick init with defaults
npx ccjk init --quick

# Start using CCJK
ccjk
```

### First Steps

```typescript
import { createTool, createVersionService, ContextManager } from 'ccjk';

// Create a code tool
const claude = createTool('claude-code');
await claude.configure({ apiKey: 'your-key' });

// Use token optimization
const contextManager = new ContextManager();
const compressed = await contextManager.compress(context);
console.log(`Saved ${compressed.compressionRatio * 100}% tokens!`);

// Manage versions
const versionService = createVersionService();
await versionService.checkVersion('claude-code');
```

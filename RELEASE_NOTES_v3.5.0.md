## 🌍 English

### What's New in v3.5.0

**🏗️ Unified Template System**
- Consolidated templates into `templates/common/` for better code reuse
- Shared output-styles, git workflows, and sixStep workflows
- Reduced code duplication between Claude Code and Codex

**🔄 Dual Code Tool Support**
- Seamless Claude Code + Codex unified configuration
- One toolkit, two powerful AI coding assistants

**✨ Enhanced sixStep Workflow**
- User approval requests in Phase 2
- Automatic timestamp acquisition
- Improved execution phase clarity

**🛡️ CCR Proxy Fix**
- Automatically removes ANTHROPIC_AUTH_TOKEN conflicts
- Smoother proxy configuration experience

### Quick Start
```bash
npx ccjk
```

## ğŸ“± Quick Reference

### Common Commands

```bash
# Setup
npx ccjk                    # Interactive menu
npx ccjk init               # Full initialization
npx ccjk update             # Update workflows

# Configuration
npx ccjk config-switch      # Switch configurations
npx ccjk config-list        # List configurations

# Tools
npx ccjk ccr                # CCR management
npx ccjk ccu                # CCUsage analytics
npx ccjk check-updates      # Check for updates

# Maintenance
npx ccjk uninstall          # Uninstall CCJK
npx ccjk restore            # Restore backup
npx ccjk --version          # Check version
```

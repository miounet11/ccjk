# 上級ガイド

本章は高度な設定、テンプレート管理、国際化、トラブルシューティングに焦点を当てています。大規模な展開、チーム協力、または深いカスタマイズが必要なユーザーに適しています。

- [設定管理](configuration.md)：増分管理、バックアップ戦略、ディレクトリ構造について学びます。
- [API プロバイダープリセット](api-providers.md)：プリセットを使用して一般的なサービスプロバイダーを迅速に設定します。
- [テンプレートと出力スタイル](templates.md)：ワークフロー、システムプロンプト、出力スタイルをカスタマイズします。
- [国際化と言語](i18n.md)：CLI と AI 出力言語戦略を設定します。
- [トラブルシューティング](troubleshooting.md)：一般的なエラーとトラブルシューティング手順。

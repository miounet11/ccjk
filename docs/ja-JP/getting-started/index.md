# クイックスタート

この章では、CCJK を使用して数分で環境を初期化する方法を紹介し、インストール前の準備、一般的なコマンド、推奨操作順序をカバーします。

## クイックナビゲーション

CCJK を初めて使用する場合は、次の手順に従うことをお勧めします：

1. **[使用ガイド](installation.md) を確認** - システム要件、使用方法、インタラクティブおよび非インタラクティブ初期化フロー、完全な設定手順を理解する
2. **コア機能に慣れる** - [機能特性](../features/) を確認して Claude Code、Codex、MCP サービス、CCR プロキシなどのコア機能を理解する
3. **CLI コマンドをマスター** - [CLI コマンド](../cli/) を参照して、さまざまなコマンドの使用方法を学習する

## 推奨ワークフロー

### 初回使用

1. `npx ccjk` または `npx ccjk init` を実行して完全な初期化を実行
2. コードツールタイプを選択（Claude Code または Codex）
3. API キーを設定するか、API プロバイダープリセットを使用
4. インストールする MCP サービスを選択
5. ワークフローテンプレートと出力スタイルを選択

### 既存の環境

- `npx ccjk update` を実行してワークフローとテンプレートを更新
- `npx ccjk` を使用してインタラクティブメニューを開き、増分設定を実行
- `npx ccjk config-switch` を使用して異なる設定を切り替え

> 💡 **ヒント**：操作中、特定の機能を理解する必要がある場合は、対応する機能チャプターまたは CLI コマンドドキュメントに自由にジャンプしてください。

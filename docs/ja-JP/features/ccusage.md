# ccusage 使用分析

[ccusage](https://github.com/ryoppippi/ccusage) は、ローカルの JSONL ファイルから Claude Code/Codex CLI の使用状況を分析する CLI ツールで、非常に高速で情報量の多いトークン使用量とコスト分析を提供します。

## ccusage とは

ccusage は、開発者が AI 支援開発のコストを理解し最適化するのを助けるために設計された強力な使用分析ツールです。ローカルログファイルを解析することで、詳細な統計レポートを提供します。

## 主な機能

- **📊 多次元レポート**：日次、週次、月次の使用レポート
- **💰 コスト分析**：トークン消費量と対応するドルコストを正確に計算
- **🤖 モデル内訳**：異なるモデル（Claude Opus/Sonnet/Haiku）の使用比率を詳細に表示
- **📈 トレンド分析**：使用量の変化傾向を可視化
- **🔌 簡単な統合**：他のツールとの統合が容易な JSON/CSV エクスポートをサポート
- **🚀 ステータスバー対応**：CCometixLine と統合してリアルタイムのステータスバー表示を実現

## 使用ガイド

ccusage は CCJK に完全に統合されており、CLI コマンドを通じて直接使用できます。

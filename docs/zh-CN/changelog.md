## 版本说明

### 版本号规则

- **主版本号 (X.0.0)**: 不兼容的 API 变更
- **次版本号 (0.X.0)**: 向后兼容的功能新增
- **修订号 (0.0.X)**: 向后兼容的问题修复

### 更新类型

- **新增 (Added)**: 新功能
- **改进 (Changed)**: 现有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: Bug 修复
- **安全 (Security)**: 安全相关修复

### 升级指南

#### 从 0.4.x 升级到 0.5.x

```bash
# 1. 更新 CCJK
npm update -g ccjk

# 2. 更新项目配置
npx ccjk update

# 3. 检查权限配置（新增了更多权限类型）
npx ccjk permissions validate
```

#### 从 0.3.x 升级到 0.4.x

```bash
# 1. 更新 CCJK
npm update -g ccjk

# 2. 迁移权限配置
# 旧格式:
#   permissions: { allowWrite: true }
# 新格式:
#   permissions:
#     file:
#       write: allow

npx ccjk migrate --from 0.3 --to 0.4
```

#### 从 0.2.x 升级到 0.3.x

```bash
# 1. 更新 CCJK
npm update -g ccjk

# 2. 技能文件需要更新格式
# 旧格式: .claude/commands/
# 新格式: .claude/skills/

npx ccjk migrate --from 0.2 --to 0.3
```

## 🚀 快速开始

### 1. 环境准备

确保您的开发环境满足以下要求：

- **Node.js**: >= 22
- **pnpm**: >= 10.17.1
- **Git**: 最新版本
- **IDE**: 推荐 VS Code（带 TypeScript 支持）

### 2. 克隆和设置

```bash
# Fork 并克隆仓库
git clone https://github.com/YOUR_USERNAME/ccjk.git
cd ccjk

# 安装依赖
pnpm install

# 验证安装
pnpm test:run
pnpm typecheck
```

### 3. 开发流程

```bash
# 开发模式（使用 tsx，支持热重载）
pnpm dev

# 运行测试
pnpm test

# 代码检查
pnpm lint
pnpm typecheck

# 构建
pnpm build
```

## 🎯 开发工作流

### TDD（测试驱动开发）

CCJK 严格遵循 TDD 方法：

1. **Red**: 先写失败的测试
2. **Green**: 写最小代码使测试通过
3. **Refactor**: 重构代码，保持测试通过

**要求**：所有新功能必须先写测试！

### 代码质量标准

- **类型安全**：严格的 TypeScript 类型检查
- **代码风格**：@antfu/eslint-config 自动格式化
- **测试覆盖**：80% 最低覆盖率要求
- **文档完整**：代码注释、API 文档、用户文档

### 国际化（i18n）

- 所有用户可见的字符串必须使用 i18n
- 支持中英文两种语言
- 使用命名空间组织翻译
- 代码注释和文档使用英文

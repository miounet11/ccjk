## Getting Started

### Launching the Menu

Simply run:
```bash
ccjk
```

Or explicitly:
```bash
ccjk menu
```

### First-Time Experience

When you launch CCJK for the first time, you'll see:

1. **Welcome Screen** - Brief introduction
2. **Quick Setup Wizard** - Get configured in 3 minutes
3. **Main Menu** - Your command center

### System Requirements

- Node.js 16+ or Bun
- Terminal with color support
- 80+ column width (recommended)

## Troubleshooting

### Problem: Menu doesn't display correctly

**Solution**:
1. Ensure terminal width is 80+ columns
2. Check terminal supports colors
3. Update terminal emulator
4. Try: `ccjk menu --no-color`

### Problem: Can't connect to API

**Solution**:
1. Run Option 11 (Diagnostic Tools)
2. Check API key is valid
3. Verify network connection
4. Check firewall settings
5. Try different provider

### Problem: MCP services won't install

**Solution**:
1. Check Node.js version (16+)
2. Verify npm/pnpm installed
3. Check disk space
4. Run with sudo if needed
5. Check service requirements

### Problem: Slow performance

**Solution**:
1. Clear cache (Option 10)
2. Reduce MCP services
3. Check network speed
4. Update CCJK
5. Check system resources

### Problem: Language not switching

**Solution**:
1. Use Option 15 explicitly
2. Check config file
3. Restart CCJK
4. Clear cache

### Problem: Lost configuration

**Solution**:
1. Check `~/.ccjk/config.json`
2. Run Option 1 (Quick Initialize)
3. Restore from backup
4. Contact support

## FAQ

### Q: How do I update CCJK?

**A**: Use Option 16 (Check Updates) or run:
```bash
npm update -g ccjk
# or
bun update -g ccjk
```

### Q: Can I use multiple API providers?

**A**: Yes! Configure multiple providers with Option 2, then switch between them with Option 7.

### Q: How much does CCJK cost?

**A**: CCJK is free and open-source. You only pay for API usage from your chosen provider.

### Q: What's the difference between code tools?

**A**: Each tool has different strengths:
- **Claude Code**: Best overall, official Anthropic
- **Cursor**: Great IDE integration
- **Aider**: Excellent for file editing
- **Continue**: Fast and lightweight
- **Cline**: Simple and easy
- **Codex**: OpenAI-powered

### Q: Can I create custom workflows?

**A**: Yes! Use Option 6 to access the workflow marketplace, then customize or create your own.

### Q: How do I backup my configuration?

**A**: Your config is in `~/.ccjk/config.json`. Copy this file to backup.

### Q: Can I use CCJK in CI/CD?

**A**: Yes! Use non-interactive mode:
```bash
ccjk --skip-prompt --provider=anthropic --api-key=$API_KEY
```

### Q: How do I report bugs?

**A**: Use Option 14 to access support links, or visit GitHub issues.

### Q: Can I contribute to CCJK?

**A**: Absolutely! Check CONTRIBUTING.md in the repository.

### Q: What's the difference between interface language and AI output language?

**A**:
- **Interface language** (Option 15): Menu and messages
- **AI output language** (Option 10): AI responses

### Q: How do I uninstall CCJK?

**A**:
```bash
npm uninstall -g ccjk
# or
bun remove -g ccjk
```

# CCJK Assets

This directory contains visual assets for the CCJK project.

## Required Assets

| File | Description | Recommended Size |
|:-----|:------------|:-----------------|
| `logo.png` | CCJK logo | 360x360 px |
| `demo.gif` | CLI demo animation | 1400x800 px |
| `ccr-icon.png` | CCR extension icon | 120x120 px |
| `ccusage-icon.png` | CCUsage extension icon | 120x120 px |
| `cometix-icon.png` | Cometix extension icon | 120x120 px |
| `superpowers-icon.png` | Superpowers extension icon | 120x120 px |

## Creating Demo GIF

You can use tools like:
- [asciinema](https://asciinema.org/) + [agg](https://github.com/asciinema/agg) for terminal recording
- [Gifox](https://gifox.io/) (macOS)
- [ScreenToGif](https://www.screentogif.com/) (Windows)

### Recommended Demo Flow

1. Run `npx ccjk`
2. Show the interactive menu
3. Navigate through options
4. Show a feature in action (e.g., `ccjk doctor`)

## Logo Design Guidelines

- Modern, clean design
- Colors: Green (#00DC82) as primary
- Include "CCJK" text or initials
- Works well on both light and dark backgrounds

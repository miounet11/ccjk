## Conflict Resolution

1. **Warn the user**: Show compatibility warning
2. **Suggest alternatives**: Recommend compatible combinations
3. **Blend intelligently**: If user insists, blend styles with primary taking precedence
4. **Allow override**: Let user force any combination

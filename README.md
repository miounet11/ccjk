## ✨ 功能

- 🚀 **快速开始** - 一键初始化、更新、健康检查
- 🔔 **任务通知** - 桌面推送通知，支持多平台
- 🧩 **扩展中心** - 集成 CCR、CCUsage、Cometix、Superpowers
- 🏪 **MCP 市场** - 一键安装 MCP 服务器
- ⚙️ **配置管理** - MCP、记忆、权限、上下文管理
- 🔧 **环境诊断** - 工作目录权限检测与修复

<br/>

## 📦 安装

```bash
# 交互式运行 (推荐)
npx ccjk

# 全局安装
npm install -g ccjk

# pnpm
pnpm add -g ccjk
```

<br/>

## 🎯 使用

```bash
# 启动交互式菜单
ccjk

# 直接执行子命令
ccjk init          # 初始化
ccjk update        # 更新 Claude Code
ccjk doctor        # 健康检查
ccjk workspace     # 环境诊断
ccjk notify        # 配置任务通知
```

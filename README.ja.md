## 🚀 クイックスタート

### ワンライナーインストール

```bash
npx ccjk
```

これだけです！CCJKがすべてを自動処理します。

### 主要コマンド

```bash
ccjk                  # インタラクティブメニュー
ccjk setup            # 初回セットアップウィザード
ccjk init             # 設定の初期化
ccjk doctor           # 環境ヘルスチェック
ccjk versions         # 全バージョン確認
ccjk upgrade          # 一括アップグレード
ccjk shencha start    # コード監査開始
```

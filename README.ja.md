### 🧠 インテリジェントスキルシステム

**本当に記憶し学習するAIアシスタント。**

```yaml
# スキルは操作に基づいて自動的にアクティブ化
triggers:
  - pattern: "*.test.ts"      # テスト中？テスト戦略をロード
  - context: "git:staged"     # コミット中？コードレビューを実行
  - project: "react"          # Reactプロジェクト？ベストプラクティスをロード
```

## 🎮 コマンドリファレンス

### 基本コマンド

```bash
ccjk                    # インタラクティブメニュー（ここから始めよう！）
ccjk init               # プロジェクト初期化
ccjk doctor             # ヘルスチェック
ccjk help               # ヘルプセンター
```

### 開発コマンド

```bash
ccjk commit             # スマートgitコミット
ccjk interview          # インタビュー駆動開発
ccjk mcp <action>       # MCPサーバー管理
ccjk browser <action>   # Agent Browser自動化
ccjk postmortem <action># 過去のバグから学ぶ
ccjk skills <action>    # AIスキル管理
ccjk hooks <action>     # ライフサイクルフック設定
```

### 設定コマンド

```bash
ccjk config <action>    # 設定管理
ccjk providers <action> # APIプロバイダー管理
ccjk permissions        # 権限管理
ccjk sandbox <action>   # サンドボックス環境制御
```

### システムコマンド

```bash
ccjk system setup       # 初回セットアップ
ccjk system upgrade     # 全コンポーネントをアップグレード
ccjk system versions    # バージョン確認
ccjk check-updates      # アップデート確認
ccjk stats              # 使用統計
ccjk session <action>   # セッション管理
```
